Core Modules

Feature Overview

Six production-ready modules covering the complete equipment service lifecycle.

📋

Module 01

Call Sheet Management

Recorders log incoming hospital service requests with equipment details, urgency, and contact info. Each call sheet triggers the entire downstream service workflow.

📸

Module 02

Assessment Reports + Photo Upload

Technicians file detailed on-site assessments — scope of work, warranty status, parts list, cost estimates — with multi-image upload powered by Dropzone.js.

✅

Module 03

Multi-Role Approval Workflow

Assessments pass through a 3-stage sign-off chain: Service Manager validates scope, Inventory Manager confirms parts availability, Marketing finalises pricing.

📦

Module 04

Job Order Lifecycle

Approved assessments become Job Orders. Managers attach Purchase Orders (or flag P.O. to Follow), assign technicians, and track work through to photo-verified completion.

📅

Module 05

FullCalendar Scheduling

Visual calendar for scheduling technician site visits and job orders. FullCalendar.js provides clear date-based dispatch management across the entire service team.

📊

Module 06

Chart.js Analytics Dashboard

Admin and Manager dashboards surface real-time charts for call sheet volume, job order status distribution, and assessment history — all driven by Chart.js.

Access Control

6-Role RBAC System

Each role lands on its own dashboard after login — redirect logic is baked into the authentication layer via ASP.NET Core Identity.

👑

Admin

Full system access. Manages all users, all modules, and system configuration.

📞

Recorder

Logs incoming hospital service requests and creates Call Sheets.

👔

Service Manager

Assigns technicians, approves assessments, creates and tracks Job Orders.

🔧

Technician

Files Assessment Reports, uploads on-site photos, and closes Job Orders.

📦

Inventory Manager

Validates parts and materials availability during the approval workflow.

💰

Marketing

Handles price estimation and billing as the final approval stage.

Service Lifecycle

End-to-End Workflow

Six stages from hospital call to confirmed job completion.

1

Hospital Call

Hospital contacts the service company about faulty equipment

2

Call Sheet

Recorder logs equipment, hospital details, and urgency level

3

Site Assessment

Technician visits hospital, files report with scope, parts, and photos

4

3-Stage Approval

Manager, Inventory, and Marketing each sign off sequentially

5

Job Order

Manager creates Job Order and attaches or flags the Purchase Order

6

Completion

Technician finishes work, uploads proof photos, closes the order

Built with ASP.NET Core 8, SQL Server, and 6 integrated frontend libraries.
